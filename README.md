# Local Plan

Local Plan is a minimal local desktop task list.

## What it does

- Add a task manually.
- Add optional subtask notes under a task.
- Edit task text and subtask notes after creating it.
- Click a task button to mark it done.
- Restore a completed task if needed.
- Clear completed tasks.
- Keep tasks locally in the app storage.
- Press `Ctrl + Alt + Space` to show the app while it is running.
- Use the floating `LP` button to open the app, drag it around the screen, and release to snap it to the nearest screen edge.

No categories, no reminders, no due dates, no preset tasks.

## Commands

```bash
npm install
npm run desktop:dev
npm run package:win
```
