# Local Plan

Local Plan is a minimal local desktop task list.

## What it does

- Add a task manually.
- Add optional subtask notes under a task.
- Edit task text and subtask notes after creating it.
- Delete an individual task after confirming.
- Drag each task's progress slider from 0% to 100%.
- Task cards change color as progress increases.
- Progress never marks a task done by itself; use the task button to complete it.
- Click a task button to mark it done.
- Restore a completed task if needed.
- Clear completed tasks.
- Add and delete title-and-content notes in a separate Notes view.
- Keep tasks and notes locally in the app storage.
- Press `Ctrl + Alt + Space` to show the app while it is running.
- Use the floating `LP` button to open the app, drag it around the screen, and release to snap it to the nearest screen edge.
- Show or hide the floating `LP` button from the main window.

No categories, no reminders, no due dates, no preset tasks.

## Commands

```bash
npm install
npm run desktop:dev
npm run package:win
```
